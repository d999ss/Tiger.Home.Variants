## **🎨 Key Features & Technologies**

### **Design System:**
- **Framework:** Next.js 15 with TypeScript
- **Styling:** Tailwind CSS with custom design tokens
- **Animations:** Framer Motion for scroll animations
- **UI Components:** Shadcn/ui component library
- **Typography:** Custom fonts (Neuropa for display, Archivo for body)

### **Navigation Features:**
- **Mega Menu System:** 4 full-width dropdown menus with featured cards
- **Sticky Header:** Auto-hide on scroll down, show on scroll up
- **Light/Dark Variants:** Transparent on homepage, solid on other pages
- **Mobile Responsive:** Full mobile menu with accordions

### **Content Features:**
- **Hero Sections:** Full-screen video backgrounds with CTAs
- **Section Components:** Reusable Hero, Section, Card components
- **Image Optimization:** Next.js Image component with Unsplash integration
- **Responsive Grid Layouts:** Tailwind grid system

### **Special Sections:**
- **Leadership Team:** Dr. Sarah Chen (CSO) profile with bio and credentials
- **Medical Advisory Board:** 4 advisors with specialties and institutions
- **Donation Hub:** Comprehensive 6-page story about tissue donation ethics
